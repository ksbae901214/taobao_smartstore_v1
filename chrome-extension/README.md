# 🛒 타오바오 상품 수집기 - 크롬 확장 프로그램

타오바오/티몰 상품 페이지에서 **버튼 한 번**으로 상품 정보를 추출하여 스마트스토어 시스템으로 전송하는 크롬 확장 프로그램입니다.

---

## 📦 설치 방법

### 1단계: 확장 프로그램 폴더 준비
`chrome-extension` 폴더를 컴퓨터에 저장합니다.

### 2단계: 크롬 확장 프로그램 페이지 열기
1. Chrome 브라우저 주소창에 입력: `chrome://extensions/`
2. 또는 **메뉴 → 도구 더보기 → 확장 프로그램**

### 3단계: 개발자 모드 활성화
- 페이지 우측 상단의 **"개발자 모드"** 토글을 켜기

### 4단계: 확장 프로그램 로드
1. **"압축해제된 확장 프로그램을 로드합니다"** 클릭
2. `chrome-extension` 폴더 선택
3. 확장 프로그램이 설치됨!

![설치 완료](https://via.placeholder.com/600x200?text=Extension+Installed)

---

## 🚀 사용 방법

### 방법 1: 플로팅 버튼 사용 (추천)

1. **타오바오 상품 페이지**로 이동
   - 예: `https://item.taobao.com/item.htm?id=656106621079`

2. 페이지 우측 하단의 **📥 버튼** 클릭

3. 상품 정보가 자동으로:
   - ✅ 클립보드에 복사됨
   - ✅ 서버로 전송됨 (서버 실행 시)

### 방법 2: 팝업 사용

1. 타오바오 상품 페이지에서 **확장 프로그램 아이콘** 클릭

2. **"현재 페이지에서 상품 정보 추출"** 버튼 클릭

3. 추출된 정보 미리보기 확인

4. **"서버로 전송하기"** 또는 **"JSON 복사하기"** 선택

## 📋 추출되는 정보

| 항목 | 설명 |
|------|------|
| 🏷️ 상품명 | 중국어 원문 |
| 💰 가격 | 위안화 가격, 가격 범위 |
| 🖼️ 썸네일 이미지 | 메인 상품 이미지들 |

| 📊 판매량 | 판매 수량 |

## ⚙️ 서버 연동 설정

### 서버 주소 설정
1. 확장 프로그램 팝업 열기
2. **서버 주소** 입력란에 주소 입력
   - 로컬: `http://localhost:3000`
   - 원격: `http://your-server.com:3000`
3. 연결 상태 확인 (✅ 서버 연결됨)

### 서버 API 엔드포인트

```
POST /api/products/from-extension
```

요청 본문 예시:
```json
{
  "product_id": "656106621079",
  "title": "商品名称...",
  "price": 99.00,
  "images": ["https://...jpg", "https://...jpg"],
  "detail_images": ["https://...jpg"],
  "options": [
    {
      "name": "颜色",
      "values": [
        {"name": "红色", "image": "https://...jpg"},
        {"name": "蓝色", "image": null}
      ]
    }
  ],
  "source_url": "https://item.taobao.com/item.htm?id=656106621079",
  "extracted_at": "2024-01-01T00:00:00.000Z"
```

## 🔧 문제 해결

### 상품 정보가 추출되지 않아요
- 타오바오 로그인이 되어 있는지 확인
- 페이지가 완전히 로드될 때까지 기다리기
- 페이지 새로고침 후 다시 시도

### 이미지가 추출되지 않아요
- 상품 페이지에서 썸네일을 한 번씩 클릭해서 로드
- 상세 페이지 스크롤을 끝까지 내려서 이미지 로드

### 서버 연결이 안 돼요
- 서버가 실행 중인지 확인
- 서버 주소가 정확한지 확인
- CORS 설정 확인 (서버에서 허용 필요)

### 옵션 정보가 없어요
- 타오바오 페이지 구조가 변경되었을 수 있음
- 개발자 도구(F12)에서 콘솔 에러 확인
